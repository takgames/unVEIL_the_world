使い方ガイド

  1. ステータス/スキル/バフ/敵を入力すると自動計算します。
  2. 結果は「通常/平均/会心」を1本のバーで比較表示します。
  3. プリセットで設定を保存できます。プルダウン選択で即適用。
  4. 共有URLは現在の入力をURLに埋め込みます。Markdownコピーも可。
  5. 内訳で途中計算の値を確認できます。
